# Student Pass/Fail Predictor

A neural network model that predicts whether a student will pass 
or fail based on study hours and attendance percentage.

## What it does
- Takes study hours and attendance % as input
- Predicts pass or fail using a neural network classifier
- Shows pass probability percentage

## Tech stack
- Python
- scikit-learn (MLPClassifier — neural network)
- NumPy

## How to run
1. Clone the repo
2. Create virtual environment
   python -m venv .venv
3. Activate it
   .venv\Scripts\activate
4. Install dependencies
   pip install -r requirements.txt
5. Run
   python stu.py
